# Tiancheng-Atelier

| 目录 | 说明 |
| --- | --- |
| [client/](./client) | 客户端：uni-app + Vue3 + Vite + TypeScript（H5 / 微信小程序 / App） |
| [server/](./server) | 服务端：Express + TypeScript + Prisma |

## 快速开始

```bash
# 客户端
cd client && pnpm install && pnpm dev:h5

# 服务端
cd server && pnpm install && pnpm dev
```

各子项目使用说明详见各自的 README。
